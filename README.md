# E-commerce Back End 

## Purpose
This is the back end for an e-commerce website using the latest technologies. Working API was configured to use Sequelize and interact with a MySQL database. 

## Built With
* MySQL2
* Sequelize
* Express
* dotenv package
* JavaScript
